## Requirments
- Versions: 1.13-1.17

## How To Install
1. Go to Releases on the github
2. Download the latest .jar file
3. Place it into your plugin folder

## Features
- /es help command
- /es give (player) (EntityType) (amount)
- On breaking a spawner with silk touch you get the spawner back
